Algoritmos e Lógica de Programação

📌 O que é um Algoritmo?

Um algoritmo é uma sequência finita de passos organizados para resolver um problema ou executar uma tarefa.

Exemplos do dia a dia:

  • Receita de bolo
  • Manual de instruções
  • Passos para sacar dinheiro no caixa eletrônico

🧠 O que é Lógica de Programação?

Lógica de programação é a forma de organizar o pensamento para criar algoritmos corretos e eficientes.

Estruturas Básicas:

  1. Sequência
  2. Decisão (if / senão)
  3. Repetição (para, enquanto)

📊 Exemplo de Estrutura Condicional

Se nota >= 60
    aluno aprovado
Senão
    aluno reprovado

📋 Tabela Resumo

Estrutura Função
Sequencial Executa comandos em ordem
Condicional Toma decisões
Repetição Repete comandos
